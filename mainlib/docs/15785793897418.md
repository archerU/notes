# Deer SDK

统一开发工具和环境来完成开发任务。

- 工程管理平台
- SDK

## SDK

Deer SDK 是一个本地命令工具。

- Cli
- Core：小版本自动更新，大版本确认更新
- Module

SDK
- 基础命令
	- 模块管理相关：更新模块，用户登录命令
	- CLI 
- 套件固定命令
- 插件扩展命令

### Cli 基础命令

#### deer install

```
deer install <modules...>
deer i <modules...>
```

强制重装当前版本

```
deer install <modules...> --force
```

#### deer update

```
deer update [modules...]
deer up [modules...]
```

#### deer remove

```
deer remove <modules...>
deer rm <modules...>
```

#### deer list

```
deer list
deer l
```

#### deer login

```
deer login
deer logout
```

#### deer --version

```
deer --version
```

#### deer --help

```
deer --help
```
